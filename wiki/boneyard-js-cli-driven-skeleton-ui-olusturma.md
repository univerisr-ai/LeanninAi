## Ozet
boneyard-js, mevcut UI bileşenlerinizi yakalayarak eşleşen skeleton loading ekranları oluşturan CLI tabanlı bir araçtır. Geliştiricilerin skeleton ekranlarını manuel kodlamak yerine otomatik olarak oluşturmasını sağlar ve perceived performance'ı artırır.

## Ana Noktalar
- CLI komutu ile mevcut UI bileşenlerinden skeleton oluşturma
- Framework-agnostic yapı: React, Vue, Angular ve diğer framework'lerle uyumlu
- Gerçek UI elemanlarının boyut ve layout'unu otomatik kopyalama
- Loading state göstergeleri için skeleton pattern'leri
- Perceived performance iyileştirmesi: içerik yüklenirken kullanıcıya görsel feedback
- Komut satırı entegrasyonu ile geliştirme sürecine hızlı adaptasyon
- Skeleton screen'lerin erişilebilirlik ve kullanıcı deneyimi üzerindeki etkisi
