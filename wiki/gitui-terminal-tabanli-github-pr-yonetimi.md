## Ozet
gitui, GitHub pull request'lerini terminal üzerinden hızlıca yönetmeyi sağlayan küçük boyutlu bir TUI (Text-based User Interface) uygulamasıdır. Geliştiriciler için tarayıcıya geçiş yapmadan PR inceleme, onaylama ve birleştirme işlemlerini kolaylaştırır.

## Ana Noktalar
- Tamamen terminal tabanlı çalışır ve GitHub CLI (`gh`) ile entegredir.
- Hızlı, hafif ve kurulum gerektirmez; mevcut GitHub kimlik doğrulamasını kullanır.
- Klavye kısayolları (vim tarzı) ve fare ile kullanılabilirlik sunar.
- PR listeleme, detay görüntüleme, CI durumu kontrolü, dosya değişikliklerini inceleme ve onay/merge işlemleri yapılabilir.
- React 19 ve OpenTUI ile geliştirilmiştir.
